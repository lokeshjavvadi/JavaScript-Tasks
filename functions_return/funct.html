1.	Check Leap Year
–	Task: Create a function isLeapYear that takes a year as an argument and returns true if the year is a leap year, and false otherwise.
–	Example: isLeapYear(2020) should return true, isLeapYear(2021) should return false.
2.	Check Positive or Negative
–	Task: Write a function isPositive that takes a number as an argument and returns true if the number is positive, and false if it is negative.
–	Example: isPositive(5) should return true, isPositive(-3) should return false.
3.	Check Divisibility by 3 and 5
–	Task: Create a function isDivisibleBy3And5 that takes a number as an argument and returns true if the number is divisible by both 3 and 5, and false otherwise.
–	Example: isDivisibleBy3And5(15) should return true, isDivisibleBy3And5(10) should return false.
4.	Check Vowel or Consonant
–	Task: Write a function isVowel that takes a character as an argument and returns true if the character is a vowel, and false if it is a consonant.
–	Example: isVowel('a') should return true, isVowel('b') should return false.
5.	Check Uppercase or Lowercase
–	Task: Write a function isUppercase that takes a character as an argument and returns true if the character is uppercase, and false if it is lowercase.
–	Example: isUppercase('A') should return true, isUppercase('a') should return false.
